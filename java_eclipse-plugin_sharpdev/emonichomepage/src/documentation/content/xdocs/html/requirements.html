To use this plug-in you will need:

  • Eclipse 3.2 or newer (Tested on Eclipse 3.2.2, 3.3, and 3.4 development line)
  • A .NET development environment (Mono or Microsoft .Net)

If you have more than one .NET framework environments installed and wish to control which one is used, you will need:

  • NAnt (see http://nant.sourceforge.net)
