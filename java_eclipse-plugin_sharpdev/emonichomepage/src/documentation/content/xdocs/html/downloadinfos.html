There are basically two ways to install the plug-in:

  • Use the eclipse update manager
  • Download the zip file and install it by copying it into your Eclipse directory and unpacking it

Download via update site

Add one of the following update sites to Your Exlipse update configuration (Menu: Help->Software updates->Find and Install)
AdressLocation
http://emonic.sourceforge.net/updatesite/internap/site.xmlSan Jose, CA - North America
http://emonic.sourceforge.net/updatesite/nchc/site.xmlTainan, Taiwan - Asia
http://emonic.sourceforge.net/updatesite/ovh/site.xmlParis, France - Europe

Download via sourceforge

Alternativly, you can also download the software from the Sourceforge Download Page . You can find various files there ($vn: version number):
FileRemark
emonic_$vn.zipThe binary build of emonic. Unzip in Your eclipse inst to install.
emonic_src_$vn.zipThe source code. Soureceforge projects require to publish the source with the releases. However: It is tricky to build if You want to include the emonicinformator and the online help; so if You are no developer I ask You to use the binary build.
emonicinformator_$vn.zipThe emonic informator as extra package. Since it is included in the plugin as well, You normaly won't need it
org_emonic.base_updatesite_plugins_$vn.jarServed via the eclipse update manager. No use to download it by hand!
org_emonic.base_updatesite_features_$vn.jarServed via the eclipse update manager. No use to download it by hand!
